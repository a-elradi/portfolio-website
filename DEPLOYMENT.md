# Deployment Guide

## Option 1: Vercel (Recommended - Easiest & FREE)

### Steps:
1. Go to [vercel.com](https://vercel.com)
2. Sign up with GitHub
3. Click "Import Project"
4. Select your portfolio-website repository
5. Click "Deploy"
6. Your site is live! 🎉

**Add Custom Domain:**
- In Vercel dashboard: Settings > Domains
- Add your domain (purchased from Namecheap, Google Domains, etc.)
- Update DNS records in your domain provider

**Cost:** FREE (with generous free tier)

## Option 2: Netlify

### Steps:
```bash
npm run build && npm run export
```
Then drag `/out` folder to netlify.com

**Cost:** FREE

## Domain Names

Cheap options:
- Namecheap ($0.88/year first year)
- Google Domains ($12/year)
- Vercel Domains (integrated)
